# AEScryptor Tool Description
Encrypt and Decrypt files with AES-128 (16bytes key).

# Usage
[1] Choose [1] or [0] ( [Encryption] or [Decryption] )

[2] Type keyphrase!

[3] Select a file (or path of a file)

[4] Enjoy!
